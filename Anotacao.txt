
Criar Migration

Se você estiver usando .NET CLI:

dotnet ef migrations add RenameColumn

Se você estiver usando Package Manager Console:

Add-Migration RenameColumn

Se você estiver usando .NET CLI:

dotnet ef database update

Se você estiver usando Package Manager Console:

Update-Database
